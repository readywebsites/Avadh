Key Features

3D Tilt Effects

Real-time mouse tracking with dynamic rotateX and rotateY transforms. Cards follow your cursor position creating an immersive 3D experience.

  • Mouse position tracking
  • Dynamic tilt angles
  • Smooth transitions
  • Customizable max rotation

Parallax Depth

Multi-layered parallax animations create stunning depth perception. Different elements move at different speeds for realistic 3D effect.

  • Stars (50% parallax)
  • Text content (70% parallax)
  • Profile info (30% parallax)
  • Smooth layer transitions

Star Animations

Animated star ratings with pulse effects. Stars pop in with bounce easing when the card becomes visible on screen.

  • Scroll reveal animation
  • Staggered star appear
  • Pulse glow effect
  • Customizable timing

Badge System

Verified guest badges with automatic animation. Pop-in effect on hover makes verification status prominent and trustworthy.

  • Auto-reveal on hover
  • Scale animation
  • Customizable styling
  • Accessibility support

Fully Responsive

Optimized for all screen sizes with intelligent breakpoints. Desktop gets full effects, mobile gets simplified touch-friendly experience.

  • Desktop: Full 3D effects
  • Tablet: Reduced tilt angle
  • Mobile: Minimal tilt (4deg)
  • Touch optimized

Accessibility First

WCAG compliant with keyboard navigation, screen reader support, and respects user motion preferences for inclusive experience.

  • Keyboard navigation (Tab, Enter)
  • ARIA labels with ratings
  • prefers-reduced-motion support
  • Dark mode ready

Implementation Guide

HTML Structure

Link CSS and load the review section:

<link href="css/review-cards-3d.css" rel="stylesheet"> <section class="reviews-section"> <div class="reviews-grid"> <article class="review-card"> <div class="review-card-inner"> <div class="review-glow"></div> <div class="review-light-ray"></div> <div class="review-stars"> <span class="star filled"><i class="bi bi-star-fill"></i></span> ... </div> <p class="review-text">Your review...</p> <div class="reviewer-profile"> ... </div> </div> </article> </div> </section>
CSS Classes

Main styling classes available:

.reviews-section - Main container .reviews-grid - Grid layout .review-card - Card wrapper (perspective) .review-card-inner - Inner card (transforms) .review-glow - Background glow effect .review-light-ray - Sweep animation .review-stars - Star rating container .review-text - Review text .reviewer-profile - Profile section .review-verified - Verification badge .reviewer-avatar - Avatar image wrapper .reviewer-name - Guest name .reviewer-title - Guest type

Customization: Edit variables at top of review-cards-3d.css

JavaScript Engine

Initialize the tilt engine:

<script src="js/review-tilt-engine.js" defer></script> // Auto-initializes on DOMContentLoaded with default settings: // maxTilt: 15 degrees // scale: 1.05x // glowIntensity: 0.8 // Access via window.tiltReviewEngine console.log(window.tiltReviewEngine); // Listen for clicks document.addEventListener('reviewCardClicked', (e) => { console.log('Card clicked:', e.detail.index); console.log('Data:', e.detail.card); });
Available Methods
// Get card data const data = tiltReviewEngine.getCardData(0); // Returns: { text, rating, name, title } // Enable magnetic effect tiltReviewEngine.enableMagneticEffect(card); // Animate counter tiltReviewEngine.animateCounter(element, targetNumber, duration); // Reset card to default state tiltReviewEngine.resetCardTransform(card); // Cleanup on page unload tiltReviewEngine.destroy();

Configuration:

const engine = new TiltReviewEngine({ maxTilt: 15, // Max rotation angle scale: 1.05, // Hover scale shadowAmount: 30, // Shadow depth glowIntensity: 0.8 // Glow opacity });
