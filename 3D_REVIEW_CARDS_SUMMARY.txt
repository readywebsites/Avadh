ğŸš¨ TROUBLESHOOTING QUICK FIXES
================================================================================

ISSUE: Cards not showing 3D tilt
FIX: Check browser console (F12) for errors
     Verify CSS & JS files are linked
     Clear browser cache and refresh

ISSUE: Parallax not visible
FIX: Verify HTML elements exist in template
     Check CSS is loaded
     Look for console errors

ISSUE: Mobile still shows full tilt
FIX: Clear browser cache completely
     Hard refresh (Ctrl+Shift+R)
     Check media queries in CSS

ISSUE: Performance is sluggish
FIX: Check DevTools Performance tab
     Verify GPU acceleration enabled
     Reduce number of particles
     Check for console memory leaks

ISSUE: Dark mode not working
FIX: Check browser dark mode setting
     Verify CSS has dark mode rules
     Test in different browser

ISSUE: Animations not smooth
FIX: Check if will-change is set
     Verify GPU acceleration active
     Test on different device
     Check browser performance mode
