## ğŸ”§ Troubleshooting Quick Reference

### Issue: Animations not playing
**Solution:** 
1. Check GSAP loads: Type `gsap` in console
2. Verify CSS loads: Check Network tab for 200 status
3. Clear cache: Ctrl+F5 (hard refresh)

### Issue: Cards not responsive
**Solution:**
1. Verify viewport meta tag exists
2. Check grid columns adjust on resize
3. Open DevTools responsive mode

### Issue: Performance issues
**Solution:**
1. Record in Performance tab
2. Check for layout thrashing
3. Disable animations: `gsap.globalTimeline.pause()`

### Issue: Colors not showing
**Solution:**
1. Check CSS custom properties: `:root { --color-gold: #color; }`
2. Clear browser cache
3. Verify hex color format
