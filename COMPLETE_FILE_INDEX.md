## üé® CUSTOMIZATION

### Change Colors
```css
:root {
  --primary: #978667;      /* Your color */
  --accent: #c9a227;       /* Your color */
  --light: #ebd7b2;        /* Your color */
}
```

### Change Animation Speed
```css
:root {
  --timing-fast: 0.3s;
  --timing-normal: 0.6s;
  --timing-slow: 1s;
}
```

### Change Easing
Edit easing functions in `modern-gsap-animations.css` CSS variables.
