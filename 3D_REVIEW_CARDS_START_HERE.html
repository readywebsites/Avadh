üíª JavaScript API

Auto-Initializes on Page Load

The review card engine starts automatically. Access via window.tiltReviewEngine

  • Get Data: tiltReviewEngine.getCardData(0) ‚Üí Returns card info
  • Listen Events: document.addEventListener('reviewCardClicked', ...)
  • Magnetic Effect: tiltReviewEngine.enableMagneticEffect(card)
  • Animate Counter: tiltReviewEngine.animateCounter(element, 1200)
  • Custom Options: Pass config object to new TiltReviewEngine({...})
